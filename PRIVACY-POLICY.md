# Privacy Policy for TopstepX SL/TP Assistant

**Last updated:** December 11, 2024  
**Version:** 4.4.1

## Overview

TopstepX SL/TP Assistant ("the Extension") is committed to protecting your privacy. This policy explains how the Extension handles your information.

## Data Collection

**The Extension does NOT collect, store, or transmit any personal data to external servers.**

### What We Don't Collect:
- ❌ Personal information (name, email, address)
- ❌ Trading data or account information
- ❌ Browsing history
- ❌ Financial information
- ❌ Usage analytics or telemetry
- ❌ Cookies or tracking data

## Local Storage

The Extension uses Chrome's local storage API to save your preferences:

### What is Stored Locally:
- ✅ Configuration settings (colors, line width, etc.)
- ✅ Risk management preferences
- ✅ Display options

### Where Data is Stored:
- **Location:** On your device only
- **Access:** Only by the Extension
- **Transmission:** Never sent to external servers
- **Deletion:** Removed when you uninstall the Extension

## Permissions Used

The Extension requires the following permissions:

### 1. `storage`
**Purpose:** Save your configuration preferences locally  
**Data Access:** Only extension settings  
**Scope:** Local to your device

### 2. `activeTab`
**Purpose:** Access TopstepX.com to display visual lines on charts  
**Data Access:** Only chart display area  
**Scope:** Only when you're on TopstepX.com

### 3. `scripting`
**Purpose:** Inject visual elements (lines) into TopstepX charts  
**Data Access:** Chart rendering only  
**Scope:** Limited to visual display

### 4. Host Permissions: `https://topstepx.com/*`
**Purpose:** Access TopstepX charts to draw lines  
**Data Access:** Chart interaction only  
**Scope:** Only TopstepX.com

## Third-Party Services

### TopstepX
The Extension operates on TopstepX.com to access and modify chart displays. No data is sent to TopstepX beyond normal website interaction.

### TradingView
TopstepX uses TradingView charts. The Extension interacts with these charts locally in your browser to draw lines. No data is sent to TradingView.

## Data Security

- All configuration data is stored using Chrome's secure storage API
- No data transmission over the internet
- No external API calls
- No third-party analytics or tracking
- Open source code available for inspection

## Your Rights

You have the right to:
- **Access:** View all stored data in Chrome's extension storage
- **Delete:** Remove all data by uninstalling the Extension
- **Control:** Modify all settings through the Extension popup

### How to Delete Your Data:
1. Uninstall the Extension from Chrome
2. All local data is automatically deleted
